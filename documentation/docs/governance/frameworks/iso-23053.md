# ISO/IEC 23053

## In one sentence

ISO's reference architecture and life-cycle for AI/ML systems —
vocabulary and structural decomposition, not a management system.

## What it adds

23053 names the components of an AI/ML system and the roles of the
stakeholders around it. Useful for organisations that want a shared
conceptual model across engineering, governance, and audit.
